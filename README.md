## 🛠 Sơ đồ kết nối (Nút)

| Chân | Chân GPIO | Định nghĩa Code |
| :--- | :---: | :--- |
| **Button UP** | 4 | `PIN_BTN_UP` |
| **Button DOWN** | 3 | `PIN_BTN_DOWN` |
| **Button SELECT** | 2 | `PIN_BTN_SEL` |
| **Button BACK** | 1 | `PIN_BTN_BACK` |
## 🛠 Sơ đồ kết nối (Màn hình SSD1306 I2C 0,96")
| Chân | Chân GPIO | Định nghĩa Code |
| :--- | :---: | :--- |
| **SDA** | 5 | `SDA` |
| **SCL** | 6 | `SCL` |

> **Lưu ý:** Đảm bảo các chân nút nhấn được nối với chân `GND`.
